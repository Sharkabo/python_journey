# Task Description

## Goal 1
Ask the user to input a **temperature** (in Celsius).

## Goal 2
Write logic to print advice based on the temperature:
- If it is **greater than 30**, print "It's hot, drink water."
- If it is **between 15 and 30** (inclusive), print "Nice weather."
- If it is **less than 15**, print "Bring a jacket."

*Hint: You can use `if`, `elif`, and `else`.*

---
**Expected Output (Example):**
```text
Enter temperature: 10
Bring a jacket.
```
