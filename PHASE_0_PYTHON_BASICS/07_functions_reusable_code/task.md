# Task Description

## Goal 1
Define a function called `check_password` that takes one parameter `input_pass`. Inside the function, use an `if` statement to check if `input_pass` equals "secret123". If yes, print "Access Granted", otherwise print "Access Denied".

## Goal 2
Call your function twice to test it:
1. Once with the wrong password.
2. Once with "secret123".

---
**Expected Output:**
```text
Access Denied
Access Granted
```
