# Task Description

## Goal 1
Create a dictionary called `book` with these keys:
- "title": "Harry Potter"
- "author": "J.K. Rowling"
- "year": 1997

## Goal 2
Print a sentence using the dictionary values:
"Harry Potter was written by J.K. Rowling."

## Goal 3
Change the "year" to 2000 (just to see how to update it).
*Hint: `book["year"] = 2000`*

---
**Expected Output:**
```text
Harry Potter was written by J.K. Rowling
```
