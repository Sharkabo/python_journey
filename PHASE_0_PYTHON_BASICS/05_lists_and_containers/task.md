# Task Description

## Goal 1
Create a list called `todo_list` with two tasks already inside: "Buy Milk", "Walk Dog".

## Goal 2
Ask the user to "Enter a new task:".

## Goal 3
Add (append) the user's new task to the `todo_list`.

## Goal 4
Print the **first** task and the **last** task (the one just added).
*Hint: You can access the last item using index `-1` or index `2`.*

---
**Expected Output:**
```text
Enter a new task: Learn Python
First task: Buy Milk
Last task: Learn Python
```
