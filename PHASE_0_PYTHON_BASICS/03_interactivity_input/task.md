# Task Description

## Goal 1
Ask the user for their **name** using `input()` and store it in a variable.

## Goal 2
Ask the user for their **birth year** (e.g., 1990) using `input()` and store it.

## Goal 3
Calculate their approximate age (assuming current year is 2026).
*Hint: Remember to convert the birth year to an integer before subtracting!*

## Goal 4
Print a message like: "Ian, you are approx 36 years old."

---
**Expected Output:**
```text
What is your name? Ian
What year were you born? 1990
Ian, you are approx 36 years old.
```
