# Task: Build a Binary Tree

Open `answer.py` in this folder and complete the following objectives:

## Goal 1: Create TreeNode Class
Create a `TreeNode` class with:
- `value`, `left`, and `right` attributes
- Proper initialization

## Goal 2: Create BinaryTree Class
Create a class that:
- Has a `root` attribute
- Can build a tree manually
- Provides helper methods

## Goal 3: Implement Tree Operations
Add these methods:
- `height()` - calculate tree height
- `count_nodes()` - count total nodes
- `count_leaves()` - count leaf nodes

---

**Expected Output:**
```text
Tree height: 3
Total nodes: 7
Leaf nodes: 4
```
