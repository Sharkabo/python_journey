# Task: Build a Singly Linked List

Open `answer.py` in this folder and complete the following objectives:

## Goal 1: Create Node Class
Create a `Node` class with:
- `data` attribute to store value
- `next` attribute to point to next node
- Proper initialization

## Goal 2: Create LinkedList Class  
Create a `LinkedList` class with:
- `head` attribute (starts as None)
- `append(data)` method to add nodes
- `display()` method to print all nodes
- `length()` method to count nodes

## Goal 3: Test Your Linked List
Demonstrate:
- Adding multiple nodes
- Displaying the list
- Getting the length

---

**Expected Output:**
```text
List: 10 -> 20 -> 30 -> None
Length: 3
```
