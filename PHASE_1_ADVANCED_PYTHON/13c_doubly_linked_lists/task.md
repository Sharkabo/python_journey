# Task: Build a Doubly Linked List

Open `answer.py` in this folder and complete the following objectives:

## Goal 1: Create DNode Class
Create a `DNode` class with:
- `data`, `next`, and `prev` attributes
- Proper initialization

## Goal 2: Create DoublyLinkedList Class
Create a class with:
- `head` and `tail` attributes
- `append(data)` - adds to end
- `prepend(data)` - adds to beginning
- Maintains both next and prev links

## Goal 3: Implement Bidirectional Traversal
Add these methods:
- `display_forward()` - traverse from head
- `display_backward()` - traverse from tail
- Demonstrate advantages over singly linked list

---

**Expected Output:**
```text
Forward: 10 <-> 20 <-> 30 <-> None
Backward: 30 <-> 20 <-> 10 <-> None
```
