# Task: Build a Binary Search Tree

Open `answer.py` in this folder and complete the following objectives:

## Goal 1: Implement BST Insert
Create a BST class with:
- `insert(value)` method
- Maintains BST property (left < parent < right)
- Handles duplicates

## Goal 2: Implement BST Search
Add these methods:
- `search(value)` - returns True/False
- `find_min()` - returns smallest value
- `find_max()` - returns largest value

## Goal 3: Test BST Operations
Demonstrate:
- Inserting multiple values
- Searching for values
- Finding min/max

---

**Expected Output:**
```text
Inserted: 50, 30, 70, 20, 40, 60, 80
Search 40: Found
Search 100: Not found
Min: 20, Max: 80
```
