# Task Description

## Goal 1
Define a function called `check_password` that takes one input (parameter) called `input_pass`.

## Goal 2
Inside the function, use an `if` statement:
- If `input_pass` is equal to "secret123", print "Access Granted".
- Otherwise, print "Access Denied".

## Goal 3
Call your function twice to test it:
1. Once with the wrong password.
2. Once with "secret123".

---
**Expected Output:**
```text
Access Denied
Access Granted
```
