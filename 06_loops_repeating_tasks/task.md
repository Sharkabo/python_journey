# Task Description

## Goal 1
Create a list of numbers: `[10, 20, 30, 40, 50]`.

## Goal 2
Write a `for` loop that goes through each number in the list.

## Goal 3
Inside the loop, add **5** to the number and print the result.
(e.g., 10 becomes 15, 20 becomes 25...)

---
**Expected Output:**
```text
15
25
35
45
55
```
