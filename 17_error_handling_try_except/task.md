# Task Description

## Goal 1
Write a `try` block that asks the user for a number and prints it doubled. Then, write an `except` block that catches errors if they type "banana" instead of a number (i.e. catches `ValueError` errors). In the except block, print "Please enter digits only."

---
**Expected Output (Scenario 1):**
```text
Enter number: 10
20
```

**Expected Output (Scenario 2):**
```text
Enter number: hello
Please enter digits only.
```
