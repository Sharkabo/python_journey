# Task: Calculator Functions

Open `answer.py` in this folder and complete the following objectives:

## Goal 1: Create Math Functions
Create four functions that each take two numbers and return the result:
- `add(a, b)` - returns a + b
- `subtract(a, b)` - returns a - b
- `multiply(a, b)` - returns a * b
- `divide(a, b)` - returns a / b

## Goal 2: Get User Input
Ask the user to input two numbers (convert to float).

## Goal 3: Perform Calculations
Call all four functions with the user's numbers and display the results using f-strings.

---
**Expected Output:**
```text
Enter first number: 10
Enter second number: 5
10.0 + 5.0 = 15.0
10.0 - 5.0 = 5.0
10.0 * 5.0 = 50.0
10.0 / 5.0 = 2.0
```
